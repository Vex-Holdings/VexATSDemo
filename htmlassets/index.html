Demo Flow

  1. Market Summary Page - Public facing showing quote wall
  2. Register - collect basic contact info to build Magic login
  3. B/D Account Application
    • Personal Accredited
    • Corporate, LLC
  4. Link to Transfer Agent
    • Set up link to existing account
    • or, agree to send over info to set up an account
  5. Order Form with COD setup
  6. Order entry into the book
  7. Order match
  8. Match reported to clients
  9. Match reported to Transfer Agent
  10. Transfer Agent checks match report against outstanding COD instructions
    • if no good, DK settlement back to the ATS
    • if good, execute on COD instructions
      1. debit and credit certificates on MSHF
      2. debit cash certificate from buyer; credit cash certificate to seller
      3. report partial/whole completion of COD instructions
      4. report transfer complete on match report to ATS
  11. ATS receives settlement report from Transfer Agent
